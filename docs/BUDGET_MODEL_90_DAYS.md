# Budget Model (90 Days)

## Team Cost (lean)
- Unity Dev (1): $1,000 - $2,500 / month
- 2D Artist (part-time): $400 - $1,000 / month
- QA (part-time): $200 - $600 / month
- PM/Founder: owner-driven

Estimated monthly burn (lean): $1,600 - $4,100

## Tooling / Infra
- Unity: free/pro as needed
- Firebase: low in early stage
- Ad tools: mostly free setup
- Optional attribution: $0 - $300/mo
- Misc assets/sfx/plugins: $100 - $500/mo

## UA (optional, phase-based)
- Month 1: $0-$300 (organic heavy)
- Month 2: $300-$1,000
- Month 3: $1,000-$3,000 (if KPIs healthy)

## 90-Day Scenarios
### Conservative
- Total spend: $5k
- Focus: product + organic
### Balanced
- Total spend: $9k
- Focus: product + light paid UA
### Aggressive
- Total spend: $15k+
- Focus: fast scale if retention proven

## Kill/Scale Rules
- If D1 < 25% after tuning -> pause UA
- If D1 >= 35% and ARPDAU improving -> scale UA gradually
