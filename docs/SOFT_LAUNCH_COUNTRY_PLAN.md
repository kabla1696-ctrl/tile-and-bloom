# Soft Launch Country Plan

## Phase 1 (Low-CPI test)
- Philippines
- Indonesia
- Bangladesh (optional local signal)

Goal: retention + crash + funnel validation

## Phase 2 (Quality check)
- Brazil
- Mexico
- Turkey

Goal: monetization signal under moderate CPI

## Phase 3 (Scale decision)
- If KPI pass, move to Tier-1 test slices
- If KPI fail, iterate before broader rollout

## Country Evaluation Rules
- Keep country if D1 >= target and CPI acceptable
- Pause country if D1 low for 3+ days despite fixes
- Never scale UA before crash + retention baseline is healthy
