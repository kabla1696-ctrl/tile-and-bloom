# Task Ownership Matrix

## Product / Founder (You)
- Vision + prioritization
- KPI review decisions
- Budget & hiring decisions
- Release go/no-go

## Unity Developer
- Core gameplay implementation
- Level tooling + tuning hooks
- Ads/IAP/Firebase integrations
- Build pipeline health

## 2D Artist
- UI kit + tile sets
- Theme/event art
- Store creatives updates

## QA
- Regression tests
- Device matrix checks
- Bug filing + repro clarity

## Weekly Ownership Check
- Each major task must have single owner
- No task without ETA
- No blocker without mitigation owner
