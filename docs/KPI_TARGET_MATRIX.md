# KPI Target Matrix (0-90 Days)

## Day 0-30 (MVP)
- D1: 30-35%
- D7: 8-10%
- ARPDAU: $0.02-$0.04
- Crash-free: >=98%

## Day 31-60 (Soft Scale)
- D1: 35-40%
- D7: 10-12%
- ARPDAU: $0.03-$0.06
- Paying Users: 1-2.5%

## Day 61-90 (Growth)
- D1: 38-42%
- D7: 12-15%
- ARPDAU: $0.05-$0.08
- Paying Users: 2-4%
- D30: 3-5%

## Red Flags
- D1 < 25%
- Crash-free < 97%
- ARPDAU flat for 14 days
- CPI rising while D1 falling
