# 7-Day Execution Sprint Template

## Day 1 (Plan)
- Set 3 outcomes for the week
- Assign owners + ETA
- Lock “must-ship” list

## Day 2 (Build)
- Core feature progress
- QA checkpoint #1

## Day 3 (Measure)
- KPI mini review
- Kill weak experiments quickly

## Day 4 (Polish)
- UX fixes
- Level balancing patch

## Day 5 (Monetize)
- Offer/ad experiment launch
- Validate guardrails

## Day 6 (Verify)
- QA checkpoint #2
- Crash/performance check

## Day 7 (Review)
- Scorecard review
- Keep/stop/start decisions
- Next sprint backlog lock
