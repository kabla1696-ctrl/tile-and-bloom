# Launch Day Runbook

## T-24h
- Final build sanity check
- Store listing final proofread
- Monitoring dashboards ready

## T-1h
- Confirm rollout %
- Team on standby
- Incident channel ready

## T+1h
- Check crash-free %
- Check install and first sessions
- Check purchase/ad pipelines

## T+4h
- Review retention early signals
- Review review-store feedback
- Patch plan if needed

## T+24h
- Go/hold/rollback decision
- Update rollout % if healthy
