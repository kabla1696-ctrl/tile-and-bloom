# Founder Dashboard (Daily 5-minute check)

## Must-watch 6
1. DAU
2. Revenue (today)
3. D1 retention (latest cohort)
4. Crash-free %
5. ARPDAU
6. Top funnel drop-off point

## Daily Questions
- Is retention improving or declining?
- Which metric moved the most today?
- Any blocker that can kill weekly goals?
- One decision to make today?

## Escalation Triggers
- Crash-free < 97.5%
- D1 drops > 4 points in 48h
- ARPDAU drops > 25% week-over-week
- CPI up + retention down together
