# Founder Command Center

## North Star
Build a profitable, retention-first casual puzzle game with sustainable growth.

## Daily Top 5
1. DAU
2. Revenue
3. D1 retention
4. Crash-free %
5. Top blocker

## Weekly Top 5
1. D7 retention
2. ARPDAU trend
3. UA efficiency (CPI x D1 quality)
4. Live-ops impact
5. Team execution rate

## Decision Framework
- If retention weak: fix product first
- If retention strong + monetization weak: tune economy/offers
- If both strong: scale UA gradually
- If quality breaks: freeze growth, fix stability
