# Play Store Launch Checklist

## Build & Quality
- [ ] Android App Bundle (.aab)
- [ ] Min SDK + target SDK correct
- [ ] Crash-free >= 98% (internal test)
- [ ] ANR rate acceptable

## Store Listing
- [ ] App name finalized
- [ ] Short/long description
- [ ] Icon (512x512)
- [ ] Feature graphic (1024x500)
- [ ] Screenshots (phone/tablet)

## Compliance
- [ ] Privacy policy URL
- [ ] Data safety form completed
- [ ] Ads declaration enabled
- [ ] IAP products configured

## Testing
- [ ] Internal test track
- [ ] Closed test (20+ users)
- [ ] Critical bugs fixed

## Launch
- [ ] Production rollout 10%
- [ ] Monitor crashes/reviews/retention
- [ ] Gradual rollout to 100%
