# Bug Triage Rules

## Severity
- P0: Crash, data loss, pay block, startup fail
- P1: Core loop break, major progression issue
- P2: UX issue, visual defects, minor gameplay inconsistency
- P3: Nice-to-have polish

## Priority Rules
1. Fix all P0 before release
2. P1 must be fixed or mitigated before scaling UA
3. P2 grouped into weekly patch
4. P3 batched into polish cycles

## Bug Ticket Format
- Title
- Build version
- Device/OS
- Repro steps
- Expected vs actual
- Screenshot/video
- Severity/Priority
