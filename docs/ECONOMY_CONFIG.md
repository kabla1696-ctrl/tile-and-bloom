# Economy Config (v0.1)

## Currency
- Coins (soft currency)

## Rewards
- Level clear: 25-60 coins (by difficulty)
- First-time clear bonus: +20 coins
- Daily reward streak: 20/30/40/50/60/80/120
- Rewarded ad: +40 coins (cap 5/day)

## Booster Costs
- Hint: 60 coins
- Shuffle: 80 coins
- Undo: 50 coins

## Continue/Revive
- Revive once per level: 120 coins or rewarded ad

## IAP Packs
- Starter Pack: $0.99 = 500 coins
- Value Pack: $2.99 = 1800 coins
- Pro Pack: $6.99 = 5000 coins
- No Ads: $2.99

## Guardrails
- No pay-to-win gating
- At least 1 free path to progress every session
